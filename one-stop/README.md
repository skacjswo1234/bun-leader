# 소상공인 폐업 지원 웹사이트

## 📋 프로젝트 개요
소상공인 폐업 지원금 및 희망리턴패키지 안내를 위한 전문 웹사이트입니다. 최대 600만원 지원금, 철거비용 지원, 전직장려수당 등의 정보를 제공하며, 무료 상담 신청 기능을 갖추고 있습니다.

---

## ✨ 최근 업데이트 (2026-02-22)

### 🎨 고급스러운 디자인 개선
전체 웹사이트의 디자인을 프리미엄 수준으로 업그레이드했습니다:
- **색상 팔레트**: 더 깊고 세련된 네이비/골드 계열
- **타이포그래피**: 크기 확대 및 자간 최적화
- **그림자 효과**: 4단계 시스템으로 깊이감 강화
- **애니메이션**: cubic-bezier 기반 부드러운 전환
- **유리 모피즘**: backdrop-filter 효과 적용
- **여백 최적화**: 더 넓고 고급스러운 간격

### 🎯 안심 관리 서비스 섹션 - 3가지 프리미엄 버전

프로젝트에 3가지 다른 스타일의 안심 관리 서비스 섹션 버전이 추가되었습니다:

#### 버전 1: 럭셔리 카드 스타일 ✨
**파일**: `version1-luxury-cards.html`, `css/version1.css`

**특징**:
- 다크 네이비 배경에 화이트/골드 액센트
- 3D 카드 호버 효과
- 골드 그라데이션 강조
- 회전하는 아이콘 애니메이션
- 넘버링 워터마크 효과

**디자인 포인트**:
- 럭셔리하고 프리미엄한 느낌
- 명확한 시각적 위계
- 골드 컬러로 고급스러움 강조

---

#### 버전 2: 모던 미니멀리즘 🎨
**파일**: `version2-modern-minimal.html`, `css/version2.css`

**특징**:
- 화이트 배경의 깔끔한 디자인
- 정보 라벨-값 구조화
- 상단 블루 라인 애니메이션
- 아이콘 배경 색상 전환 효과
- CTA 버튼 통합

**디자인 포인트**:
- 정보 가독성 최우선
- 클린하고 모던한 느낌
- 비즈니스 전문성 강조

---

#### 버전 3: 프리미엄 글래스모피즘 💎
**파일**: `version3-glassmorphism.html`, `css/version3.css`

**특징**:
- 그라데이션 배경 + 애니메이션 효과
- 유리 질감(glassmorphism)
- 부유하는 백그라운드 shape
- 반투명 카드 디자인
- 디테일 아이템 박스 구조

**디자인 포인트**:
- 최신 트렌드 디자인
- 시각적 임팩트 극대화
- 인터랙티브한 느낌
- 프리미엄 브랜드 이미지

## 🎯 주요 기능

### 현재 구현된 기능
1. ✅ **반응형 네비게이션 헤더**
   - 고정 헤더 (스크롤 시 backdrop-filter 효과)
   - 모바일 햄버거 메뉴
   - CTA 버튼 통합

2. ✅ **희망리턴패키지 안내 섹션**
   - 배경 이미지 + 그라데이션 오버레이
   - 3개 카드 (사업정리컨설팅, 철거비용지원, 전직장려수당)
   - 프리미엄 hover 효과

3. ✅ **안심 관리 서비스 섹션** (3가지 버전 제공)
   - 지원기간, 지원대상, 대상지역 정보
   - 각 버전별 고유한 디자인 스타일
   - 반응형 그리드 레이아웃

4. ✅ **실제 이용 후기 (무한 슬라이더)**
   - 5개 실제 후기 카드
   - 자동 스크롤 애니메이션
   - hover 시 일시 정지

5. ✅ **진행 절차 (4단계)**
   - 시각적 숫자 표시
   - 단계별 아이콘
   - 연결선 애니메이션

6. ✅ **무료 상담 신청 폼**
   - 2열 그리드 레이아웃
   - 입력 필드: 이름, 연락처, 업종, 지역
   - 개인정보처리방침 동의 체크박스
   - 프리미엄 버튼 효과

7. ✅ **푸터**
   - 3단 그리드 레이아웃
   - 로고, 서비스 링크, 바로가기
   - 정부지원사업 배지

8. ✅ **UI/UX 개선**
   - Scroll to Top 버튼
   - 모바일 고정 전화 버튼
   - AOS 스크롤 애니메이션
   - 부드러운 전환 효과

## 🚀 사용 방법

### 버전 비교 및 선택

각 HTML 파일을 브라우저에서 직접 열어서 비교해보세요:

1. **메인 웹사이트 보기**:
   - `index.html` 열기 - 전체 페이지 확인

2. **버전 1 보기** (럭셔리 카드):
   - `version1-luxury-cards.html` 열기

3. **버전 2 보기** (모던 미니멀):
   - `version2-modern-minimal.html` 열기

4. **버전 3 보기** (글래스모피즘):
   - `version3-glassmorphism.html` 열기

### 메인 사이트에 적용하기

원하는 버전을 선택한 후, 해당 HTML 코드를 `index.html`의 안심 관리 서비스 섹션에 복사하고, CSS도 `css/style.css`에 통합하면 됩니다.
