# 랜딩페이지 카피라이팅 가이드

## 🎯 전환율 극대화를 위한 카피라이팅 전략

이 문서는 "폐업 안심 파트너" 랜딩페이지에 적용된 고전환율 카피라이팅 기법을 설명합니다.

---

## 📋 섹션별 카피라이팅 상세 분석

### 1. Hero Section (첫 화면)

#### 🎯 목표
- 3초 내에 타겟의 주의를 끌고 공감 형성
- 즉각적인 가치 제안 전달

#### ✍️ 핵심 카피
```
헤드라인: "폐업 비용 때문에 망설이고 계신가요?"
→ 고객의 핵심 Pain Point를 직접적으로 언급

서브카피: "소상공인시장진흥공단 희망리턴패키지 공식 연계"
→ 정식 기관 명칭 사용으로 신뢰도 극대화

가치 제안: "폐업 비용 최대 600만원 지원 가능"
→ 구체적인 숫자로 혜택 명시
```

#### 💡 적용 원칙
- **Pain Point 우선**: 고객의 고민을 먼저 언급
- **권위 신호**: 정부 기관명 전면 배치
- **구체적 숫자**: "최대 600만원" 같은 명확한 가치
- **즉시성**: "지금 바로", "3초 만에"

---

### 2. Trust Section (신뢰 구축)

#### 🎯 목표
- 정부 지원 사업의 공신력 전달
- 약어 대신 정식 명칭 사용으로 신뢰도 강화

#### ✍️ 핵심 카피
```
"소상공인시장진흥공단" (O)
"소진공" (X) ← 약어 사용 금지

"서울신용보증재단" (O)
"서울신보" (X) ← 약어 사용 금지

"중소벤처기업부" (O)
"중기부" (X) ← 약어 사용 금지
```

#### 💡 적용 원칙
- **완전한 기관명**: 약어를 절대 사용하지 않음
- **시각적 강조**: 기관 아이콘과 배지 활용
- **명확한 역할**: 각 기관이 무엇을 지원하는지 명시

---

### 3. Pain Point Section (고객 고민)

#### 🎯 목표
- 고객이 겪는 실제 문제를 구체적으로 나열
- "우리가 이해하고 있다"는 공감 형성
- 각 문제에 대한 해결책 제시

#### ✍️ 핵심 카피
```
문제 1: "철거비가 너무 부담돼요"
해결책: "철거비 지원금 최대 300만원"

문제 2: "세무·노무 정리가 복잡해요"
해결책: "전문가 무료 컨설팅 제공"

문제 3: "보증금을 못 돌려받을까 봐 걱정돼요"
해결책: "법률 지원 및 점포명도비 지원"

문제 4: "지원금 신청 방법을 몰라요"
해결책: "신청부터 수령까지 전 과정 대행"
```

#### 💡 적용 원칙
- **구어체 사용**: "~해요", "~죠?" 같은 친근한 말투
- **실제 사례**: 고객이 실제로 하는 고민을 그대로 반영
- **즉각적 해결책**: 문제와 함께 바로 해결책 제시

---

### 4. Process Section (진행 절차)

#### 🎯 목표
- 복잡하지 않다는 인식 심기
- 간단한 3단계 강조

#### ✍️ 핵심 카피
```
STEP 1: "1:1 무료 상담 신청"
- "3초 만에 신청 완료"
- "24시간 내 전문 상담사가 연락드립니다"

STEP 2: "전문가 맞춤 분석"
- "사장님의 상황을 정확히 파악"
- "최적의 지원 방안을 제시"

STEP 3: "지원금 신청 및 폐업 완료"
- "복잡한 서류 작업부터 지원금 수령까지"
- "전 과정을 대행해 드립니다"
```

#### 💡 적용 원칙
- **숫자로 단순화**: "3단계"로 간단하게
- **구체적 시간**: "24시간 내", "4~6주 소요"
- **수동태 회피**: "대행해 드립니다" (능동적 표현)

---

### 5. CTA Section (행동 유도)

#### 🎯 목표
- 마지막 결정을 유도
- 모든 의심을 해소

#### ✍️ 핵심 카피
```
헤드라인: "더 이상 혼자 고민하지 마세요"
서브카피: "폐업은 끝이 아닙니다. 새로운 시작을 위한 준비입니다."

혜택 요약:
- "폐업 비용 최대 600만원 지원"
- "전문가 1:1 무료 상담"
- "복잡한 서류 작업 대행"

버튼 문구: "지금 바로 무료 상담 예약하기"
```

#### 💡 적용 원칙
- **감성적 접근**: "혼자 고민하지 마세요"
- **긍정적 프레이밍**: "끝"이 아니라 "새로운 시작"
- **무료 강조**: "무료 상담", "무료 컨설팅"
- **즉시성**: "지금 바로"

## 🎨 버튼 문구(CTA) 최적화

### ✅ 좋은 버튼 문구 예시
```
✓ "지금 바로 무료 상담 예약하기"
✓ "3초 만에 무료 상담 신청"
✓ "폐업 비용 절감 상담 신청하기"
```

### ❌ 피해야 할 버튼 문구
```
✗ "제출" (기계적)
✗ "신청하기" (가치 없음)
✗ "문의하기" (수동적)
```

## 📱 모바일 최적화 카피라이팅

### 모바일 환경 고려사항
1. **짧고 강력한 헤드라인**
   - 모바일 화면에서 한 눈에 들어오는 길이
   - 2줄 이내로 제한

2. **터치 유도 문구**
   - "터치하여 전화하기"
   - "지금 바로 전화 상담"

3. **즉시성 강조**
   - "24시간 내 연락"
   - "당일 상담 가능"

## 🔑 신뢰도 강화 기법

### 1. 정식 명칭 사용
```
AS-IS: "소진공 지원금"
TO-BE: "소상공인시장진흥공단 희망리턴패키지 지원금"
```

### 2. 구체적 수치
```
AS-IS: "지원금 지급"
TO-BE: "최대 600만원 지원 가능"
```

### 3. 프로세스 투명성
```
AS-IS: "상담 신청 후 진행"
TO-BE: "24시간 내 전문 상담사가 연락 → 맞춤 분석 → 4~6주 내 지원금 수령"
```

### 4. 보안 강조
```
- "SSL 보안 인증"
- "개인정보 암호화 관리"
- "상담 목적으로만 사용"
```

## 💬 말투(Tone & Voice) 가이드

### 친근하면서도 전문적인 톤 유지

#### ✅ 적절한 표현
```
"사장님의 상황을 정확히 파악하고"
"함께 해결해 드립니다"
"걱정하지 마세요"
"전문가가 도와드립니다"
```

#### ❌ 피해야 할 표현
```
"귀하의 요청사항" (너무 격식)
"처리해드리겠습니다" (기계적)
"빠른 시일 내" (모호함)
```

## 📊 A/B 테스트 추천 사항

### 테스트해볼 만한 변형

#### 헤드라인
- A: "폐업 비용 때문에 망설이고 계신가요?"
- B: "폐업, 혼자 고민하지 마세요"

#### 버튼 문구
- A: "지금 바로 무료 상담 예약하기"
- B: "폐업 비용 절감 방법 알아보기"

#### 가치 제안
- A: "최대 600만원 지원"
- B: "평균 450만원 지원 (최대 600만원)"

## 🚀 전환율 향상 팁

### 1. 긴급성 추가
```
"오늘 신청하신 분들에게 우선 연락드립니다"
"상담 가능 인원: 오늘 ○명 남음"
```

### 2. 사회적 증거
```
"지난달 123명의 사장님이 상담을 받았습니다"
"평균 만족도 4.8/5.0"
```

### 3. 리스크 제거
```
"상담 후 진행 여부는 사장님이 결정하십니다"
"원하지 않으시면 언제든 취소 가능합니다"
```

## 📝 체크리스트

페이지를 최종 검토할 때 다음 사항을 확인하세요:

- [ ] 모든 정부 기관명이 정식 명칭으로 표기되었는가?
- [ ] 구체적인 숫자(금액, 기간)가 명시되었는가?
- [ ] 버튼 문구에 "무료"와 "지금"이 포함되었는가?
- [ ] 고객의 Pain Point가 명확히 언급되었는가?
- [ ] 해결책이 각 문제와 함께 제시되었는가?
- [ ] 친근하면서도 전문적인 톤이 유지되었는가?
- [ ] 모바일 환경에서 읽기 쉬운가?
- [ ] 전화번호 클릭 기능이 작동하는가?

## 🎓 참고자료

### 카피라이팅 핵심 원칙
1. **명확성 > 창의성**: 창의적인 표현보다 명확한 메시지가 우선
2. **혜택 중심**: 기능이 아닌 혜택을 강조
3. **고객 언어**: 전문 용어보다 고객이 쓰는 말 사용
4. **신뢰 우선**: 화려한 수식어보다 신뢰할 수 있는 근거
